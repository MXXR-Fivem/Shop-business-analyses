# 📊 NextBuy, From raw data to smart decisions.  
NextBuy is the result of a business research analysing the habits and tendencies in the data of a supermarket.
Using data analysis and machine learning, it present some meaningful insight on the state of the market and some ways to optimize marketing and promotion.

## ▶️ Run Locally

1. Clone the repo and go into the folder : 
```bash
git clone git@github.com:EpitechBachelorPromo2028/B-DAT-201-PAR-2-1-nextbuy-20.git
cd B-DAT-200-PAR-2-1-tardis-4.git
```

2. set up venv with python :
```bash
python3.11 -m venv venv 
source venv/bin/activate
pip install -r requirements.txt
```

3. run the tardis_eda then the tardis_model :
```bash
jupyter nbconvert --to notebook --execute --inplace notebook.ipynb
```

4. run the dashboard : 
```bash
streamlit run Dashboard.py
```
